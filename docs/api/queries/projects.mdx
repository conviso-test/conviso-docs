Returns all projects the user has access

```graphql
projects(
  page: Int = 0
  limit: Int = 0
  params: ProjectSearch
  order: OrderProjectsParams
  orderType: OrderParams
): ProjectCollection!

```

### Arguments

#### `page` ([`Int`](/api/scalars/int))

#### `limit` ([`Int`](/api/scalars/int))

#### `params` ([`ProjectSearch`](/api/inputs/project-search))

#### `order` ([`OrderProjectsParams`](/api/enums/order-projects-params))

column name to order. Ex: label

#### `orderType` ([`OrderParams`](/api/enums/order-params))

ASC or DESC

### Type

#### [`ProjectCollection`](/api/objects/project-collection)
