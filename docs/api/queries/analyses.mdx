Returns all analyses the user has access

```graphql
analyses(
  page: Int = 0
  limit: Int = 0
  params: AnalysisSearch
  order: OrderProjectsParams
  orderType: OrderParams
): AnalysisCollection!

```

### Arguments

#### `page` ([`Int`](/api/scalars/int))

#### `limit` ([`Int`](/api/scalars/int))

#### `params` ([`AnalysisSearch`](/api/inputs/analysis-search))

#### `order` ([`OrderProjectsParams`](/api/enums/order-projects-params))

column name to order. Ex: label

#### `orderType` ([`OrderParams`](/api/enums/order-params))

ASC or DESC

### Type

#### [`AnalysisCollection`](/api/objects/analysis-collection)
